Python Talk

May 5, 2021   

Imagine using an R markdown file to run Python code. Well, I am! The reticulate package includes a Python engine that allows for conversation between R and Python code chunks. The two languages are super similar and have full access to each others objects. Python chunks work exactly like chunks of R code! The code is executed, the output is printed and then included within the document.

Pandas is an extremely useful package in the world of data. Not that kind of panda! The python package Pandas!

Often, real-world data is messy, unorganized and difficult to work with. Pandas works to tidy the data, making it easy to manipulate and analyze. This package allows for gather columns to rows, rows to columns, appending columns/rows to dataframes, order rows by values and so much more!

To demonstrate my python knowledge, I will be tidying the same dataset used in my Project #2 but with Python.

import pandas as pd
affairs = pd.read_csv("https://vincentarelbundock.github.io/Rdatasets/csv/Ecdat/Fair.csv")
